Clipboard paste and drag-and-drop handlers attached to the Vditor container in Editor.jsx. Intercepts image items before Vditor's default handler. Reads the file as ArrayBuffer, encodes to base64, sends to main via save-image IPC. Main saves to ./assets/ relative to currentFile (or a temp path if unsaved). Returns the relative path, which is inserted as ![](./assets/filename.ext).
