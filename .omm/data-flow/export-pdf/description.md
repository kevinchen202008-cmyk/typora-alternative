App builds the same clean HTML document as HTML export, then calls print-to-pdf IPC with { name, html }. Main creates a hidden BrowserWindow, loads the HTML via loadURL('data:text/html,...'), waits for did-finish-load, calls webContents.printToPDF({}), and saves the PDF Buffer. The hidden window is destroyed immediately after. Prevents the live app DOM (sidebar, tab bar) from appearing in the PDF.
