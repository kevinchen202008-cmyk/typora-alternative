Theme ID string (default, github, night, github-dark, or user:name). applyTheme(themeId): (1) setTheme(id) for React state, (2) swaps <link id="app-theme"> href, (3) awaits CSS load event (500ms fallback), (4) reads --vditor-theme / --vditor-content-theme / --vditor-hl-theme CSS variables from computed style, (5) calls editorRef.current.setVditorTheme(). Saved to config on every change.
