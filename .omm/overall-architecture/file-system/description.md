Two storage areas: (1) Workspace files — user's .md files on disk, read/written via read-file/write-file IPC. Auto-save triggers 5 seconds after last content change if a file path is set. (2) App data — %APPDATA%/typora-alternative/ containing config.json, themes/ (user custom themes), and any exported assets. Images pasted/dropped into the editor are saved to ./assets/ relative to the current file.
