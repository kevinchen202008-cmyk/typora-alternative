electron/preload.js — the only communication bridge between Main and Renderer. Uses contextBridge.exposeInMainWorld to attach window.electronAPI. Exposes: 20+ ipcRenderer.invoke() wrappers (request/response), 3 ipcRenderer.send() fire-and-forget calls (window controls + rendererReady), and 14 event listeners (onMenuXxx, onFileOpened, onUpdateAvailable) that each return a cleanup function for React useEffect teardown.
