File-based i18n extension system. public/locales/index.json lists available locales (id, name, nativeName). Each locale folder contains ui.json (62 keys for React UI strings) and menu.json (~45 keys for native menu labels). Currently ships en-US and zh-Hans. The renderer fetches ui.json via fetch() and caches in memory. The main process reads menu.json via fs.readFileSync() into menuLocale on startup and on language change. New languages can be added without code changes.
