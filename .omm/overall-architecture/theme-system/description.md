File-based theme extension system. public/themes/index.json is the built-in theme registry (default, github, night, github-dark). Each theme is a standalone CSS file defining :root CSS variables: --bg-editor, --bg-sidebar, --border-subtle, --text-primary, --accent, --vditor-theme, --vditor-content-theme, --vditor-hl-theme, --shadow-modal, --update-banner-*. Active theme is applied by swapping the href of <link id="app-theme"> in index.html. User themes can be placed in %APPDATA%/typora-alternative/themes/.
