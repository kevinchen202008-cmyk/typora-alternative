Central state hub for the entire UI. Manages: tabs array (multi-document), active tab index, currentFile, isDirty, theme ID, editor mode (ir/sv/wysiwyg), sidebar open/tab, focusMode, typewriterMode, word count, find/replace visibility, settings panel visibility, custom CSS path, editor font/size, and update banner. Uses refs (contentRef, currentFileRef, activeIdxRef) for values accessed in IPC callbacks to avoid stale closures. Wires all ipcRenderer event listeners on mount via a single useEffect([]).
