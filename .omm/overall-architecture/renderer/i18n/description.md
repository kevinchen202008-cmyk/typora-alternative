React context-based i18n system. I18nContext.jsx: loads locales/index.json to get available languages, auto-detects system language via navigator.language + resolveLocale(), loads the target ui.json bundle via fetch, and returns null until ready (prevents flash). index.js: fetch-based JSON loader with in-memory cache; sync t(localeId, key, vars) for interpolation; resolveLocale() for BCP-47 best-match. Locale files live in public/locales/{lang-id}/ui.json and menu.json.
