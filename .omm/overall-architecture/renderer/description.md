The Renderer Process is a Vite + React SPA running inside Electron's Chromium context. Entry point is src/main.jsx → src/App.jsx. App.jsx owns all top-level state (tabs, theme, mode, sidebar, focus/typewriter mode). It renders six child components: Editor (Vditor wrapper), Sidebar (file tree + outline), TabBar (multi-tab), StatusBar, FindReplace, and SettingsPanel. The i18n subsystem (src/i18n/) provides a React context for string translations loaded from public/locales/. All communication with the Main process goes through window.electronAPI (the preload bridge).
