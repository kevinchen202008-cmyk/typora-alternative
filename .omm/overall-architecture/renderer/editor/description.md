Vditor wrapper component using forwardRef + useImperativeHandle. Initializes Vditor once in useEffect([]). Exposes imperative API: setValue, getValue, getHTML, focus, getMode, setVditorTheme, scrollToHeading, executeFormat. Handles image paste/drop (saves to ./assets/ via saveImage IPC). Theme changes are driven externally via setVditorTheme() called by App after the CSS <link> swap resolves. Mode changes click the Vditor toolbar button programmatically. Also implements focus-mode block highlighting and typewriter-mode scroll-to-cursor.
