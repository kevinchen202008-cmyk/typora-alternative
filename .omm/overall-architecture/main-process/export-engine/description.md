Three export paths: (1) HTML — dialog.showSaveDialog + fs.writeFileSync with vditor CDN stylesheet injected. (2) PDF — creates a hidden BrowserWindow, loads the clean HTML string into it, calls webContents.printToPDF({}) and saves the Buffer to disk; avoids capturing the live DOM. (3) DOCX — spawns pandoc with the Markdown source piped via stdin; falls back gracefully if pandoc is not installed.
