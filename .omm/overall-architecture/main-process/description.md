Node.js main process (electron/main.js, CommonJS). Responsibilities: BrowserWindow lifecycle, native menu construction (dynamic — rebuilt on theme/language change), all file I/O, config persistence (userData/config.json), update checking via GitHub API, PDF export via hidden BrowserWindow, Word export via pandoc spawn, image save, and all ipcMain handlers. Also manages theme and locale registries: loads public/themes/index.json and public/locales/{lang}/menu.json at startup and on language change.
