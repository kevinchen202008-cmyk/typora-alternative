Third-party Markdown editor library (vditor@3.x) serving as the core editing surface. Copied from node_modules to public/vditor/ via scripts/copy-vditor.js (postinstall hook) to enable local CDN mode (cdn: './vditor'). Provides three editing modes: ir (Instant Rendering — Typora-like), sv (Split View), wysiwyg. Supports math (KaTeX), diagrams (Mermaid), code highlighting, image upload. Theme is updated via vd.setTheme(vditorTheme, contentTheme, hlTheme).
