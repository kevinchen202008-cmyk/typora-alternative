Typora Alternative is a Windows Markdown editor built with Electron + Vite + React. It follows the standard Electron two-process architecture: a Node.js Main process handles OS integration (menus, file I/O, config, update checks) and a Chromium Renderer process runs the React UI. They communicate exclusively through a contextBridge Preload module. The editor surface is powered by Vditor (a rich Markdown editor library) running locally from public/vditor/. Themes and locales are file-based extension points loaded at runtime from public/themes/ and public/locales/.
