Four built-in CSS theme files: default (light), github (light, GitHub style), night (dark, VS Code–inspired), github-dark (dark, GitHub dark). Each defines :root CSS variables covering all app chrome colors, content typography, Vditor integration (--vditor-theme, --vditor-content-theme, --vditor-hl-theme), shadow, and update banner colors. Also includes .vditor-reset overrides for content typography.
