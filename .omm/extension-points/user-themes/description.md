%APPDATA%/typora-alternative/themes/*.css — user-installed theme CSS files. Discovered at startup by scanning the directory (created if it doesn't exist). Returned by get-themes IPC with file:// URLs so the renderer can set <link href="file://...">. Accessible via Settings > "Open Themes Folder" button (shell.openPath). No manifest required — just a .css file with :root variable definitions.
