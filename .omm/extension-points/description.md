Two primary extension vectors: (1) Themes — CSS files with :root variable definitions, installable as built-in (public/themes/) or user (userData/themes/). (2) Locales — JSON key-value files in public/locales/{lang-id}/, no code changes required to add a new language. A third lightweight extension is Custom CSS — any .css file the user provides, injected as a <style> element on top of the active theme.
